# Thai-accounting

2 Steps to generate accounting reports:

1.  Setup    
    - Journal types
    - Document types
    - Account groups
    - Account charts
    - Beginnings
    - Financial statements
2.  Add journals
 
The accounting reports are:
 
* Account chart
* Journal
* Ledger
* Trial
* Balance sheet
* Profit and loss account
